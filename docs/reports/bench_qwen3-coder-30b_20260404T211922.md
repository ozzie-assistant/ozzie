### Root causes
- **Raw XML emission (B12)**: Qwen intermittently emits `<function=...>` XML instead of proper tool_use JSON. This is a model-level issue, not a framework bug. Appeared in B12 store_memory but not in other tests.
- **B10 plan not executing**: submit_plan may have been called incorrectly by the LLM, or the task runner failed silently.
- **B15 connector**: The file bridge process may not have been started by the gateway's ProcessSupervisor. Needs gateway log investigation.
- **B20 timeout**: The model made 11 execute calls without writing the diagnostic file — likely looping on execute instead of using file_write.
- **B30/B31**: HTTP API doesn't expose session creation directly; proper testing requires WebSocket client.
